# Vybely

Vybely is a frontend web application developed as part of an academic project.

## Google Technologies Used
- Firebase Hosting
- Firebase CLI
- Google Cloud Platform
- Google Gemini (core AI layer)
-  Gemini API
-  Google Maps AI-Powered Place Intelligence(Indirect AI usage)
-  Google Fonts(AI- Assisted Accessibility Design)

## Other Technologies
- HTML
- CSS
- JavaScript
- Vite
- GitHub

## Features
- Responsive frontend UI
- Static web application
- Deployed using Firebase Hosting

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open the local URL in browser
